## 英雄传说系列
```dataviewjs
dv.table(
	["", "标题","标签"],
	dv.pages("#英雄传说系列")
	.sort(page => [page.No], 'asc')
	.map(page => [
		page.Icon,
		page.file.link,
		page.file.etags
	])
);
```
