### 任务
```dataviewjs
dv.table(
	["任务", "AP", "额外AP", "章节", "步骤", "奖励"],
	dv.pages('"游戏攻略/英雄传说 闪之轨迹/笔记/任务"')
	.map(quest => {
		quest.ChapterObj = dv.page(quest.Time);
		return quest;
	})
	.sort(quest => [quest.ChapterObj.No, quest.Time.subpath], 'asc')
	.map(quest => [quest.file.link, quest.BasicAP, quest.ExtraAP, quest.Time, quest.Steps, quest.Bonus])
);
```
